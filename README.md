# Mesto

## Description

A website where the user can edit an interactive form, enter any name and profession, change the avatar, add pictures to the server, delete, like.

## Stack

* HTML
* CSS
* Javascript

## Implementation

* Grid Layout
* Flexbox-layout
* Media queries
* Adaptive layout
* BEM Nested
* JavaScript functionality
* Webpack
