# Место

## О проекте

Сайт, на котором пользователь может редактировать интерактивную форму, вводить любое имя и профессию, менять аватар, добавлять картинки на сервер, удалять, лайкать.

## Стек

* HTML
* CSS
* Javascript

## Реализация

* Grid Layout
* Flexbox-вёрстка
* Медиазапросы
* Адаптивная верстка
* БЭМ Nested
* Функционал на JavaScript
* Webpack
